# SDSP Front

## 프로젝트 시작 아래 Close

- Repository Clone

```git
git clone git@192.168.0.84:Naithar01/sdspfrontrels.git
```

### Frontend

- frontend 폴더로 이동

```
cd frontend
```

- 설치

```
npm install
```

- 실행

```
npm start
```

- 접속

```
http://localhost:3000
```
