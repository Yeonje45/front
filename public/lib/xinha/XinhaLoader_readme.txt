XinhaLoader는 서버에서 추가 파일을 가져오고 로딩 메시지를 표시하는 기능만 포함된 Xinha의 하위 집합입니다.
사용자에게 시각적 피드백을 빠르게 제공하여 단순한 텍스트 영역을 보면서 아무 일도 일어나지 않는지 궁금해하지 않도록 합니다.

간단한 사용법: NewbieGuide를 따르되, XinhaCore.js 대신 XinhaLoader.js를 로드하십시오.
Xinha.loadPlugins를 호출하기 전에 xinha_editors를 정의해야 합니다! 이는 이전 버전의 NewbieGuide와 다릅니다 (1단계와 2단계가 반대입니다).
